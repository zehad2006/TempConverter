# Temperature Converter Python + Ttinker
Desktop application that converts Celsius degrees in Fahrenheit and Kelvin and viceversa.
built using Python and Ttinker library, and packaged into a Windows Executable using PyInstaller
# features
- simple GUI
- Instant Temperature Conversion
- packaged into .exe for easy sharing
# what i learned
- Building GUIs with Tkinter
- Packaging Python programs into executables
- Debugging PyInstaller issues
# packaging
app was packaged using:
pyinstaller --onefile --windowed TempConversion.py
